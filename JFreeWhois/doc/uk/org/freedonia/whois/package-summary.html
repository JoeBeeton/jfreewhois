    ServerConnector
    ServerConnector is used to connect to the whois server send the query and return a WhoisReponse.

    ServerDefinitionFinder is used to find a WhoisServerDefinition for a specified host name.
    ServerLister
    WhoisLister is used to return the full list of Whois Servers from the serverlist.xml
    Whois
    Whois is a utility class used to give easy access to the Whois Library.
    WhoisResponse  
    WhoisRunner
    WhoisRunner is used to run whois queries.
