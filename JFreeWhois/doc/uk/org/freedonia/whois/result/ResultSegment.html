uk.org.freedonia.whois.result

Class ResultSegment

  • java.lang.Object
    • uk.org.freedonia.whois.result.ResultSegment

  • public class ResultSegment
extends java.lang.Object
    ResultSegment contains information about a segment of the whois response. The ResultSegment contains both a a title and value. E.g Domain Name:BBC.MOBI . In this case the heading would be Domain Name and the value would be BBC.MOBI
